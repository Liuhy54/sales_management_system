## 项目简介
### 项目介绍

深入业务场景的企业级，基于 Vue 3 + Element-plus + Echarts + Spring Boot + JavaScript + MyBatis 的 **机会管理平台。**

用户可以基于 AI 快速制作并发布多种答题应用，支持检索和分享应用、在线答题并基于评分算法或 AI 得到回答总结；管理员可以审核应用、集中管理整站内容，并进行统计分析。
